#Love Calculator ❤️
Love Calculator is a fun app that calculates compatibility between two names. Built with HTML, CSS, and JavaScript, it dynamically generates a playful compatibility score. The project features smooth animations, a responsive design, and an interactive experience for users. Perfect for light-hearted fun!
